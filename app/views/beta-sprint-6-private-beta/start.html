Create your account

It is free to create an account for HM Land Registry’s licensed data and it takes about 5 minutes to apply. Once you have created your account, you can download all the licensed data files.

You can start downloading data immediately after creating your account, if you use it for:

  • personal use, or
  • a non-UK organisation

HM Land Registry can take up to 2 days to approve your account if you use it for a UK organisation. It takes 2 days to run security checks. In most cases, you can use your account on the same day you apply.

Before you start

Information you need to open an account

To create an account, you’ll need to tell HM Land Registry your name, email, address and phone number. You’ll also need to tell us if you’re a UK resident.

Are you using the data for a UK organisation? If you are, you'll also need to include your organisation's:

  • name
  • address
  • type
  • registration number

Working for a non-UK organisation? You’ll need to include your organisation's:

  • name
  • address
  • country of incorporation

Why do I need to open an account?

  • a licence
  • conditions of access
  • a fee

You must agree to the licence for each data file before you can download it.

Free data and data with an access fee

You do not need to pay anything to open an account and most of the licensed data is also free.

There are a small number of data files with a fee. You can see the list of the data files with a fee (and their costs) on HM Land Registry’s fees page.

The free data files are:

  • data file 1
  • data file 2

Consequences of data misuse

Data files with a licence have usage restrictions (you can use the data, but you can only use it in a certain way).

HM Land Registry asks you to create an account because of the legal consequences for data misuse. For example, if you sell the data HM Land Registry can share your account details with Royal Mail. Royal Mail may take you to court for data misuse.

! If you do not use the data according to the licence, Royal Mail may take you to court for data misuse.

Read each data licence to find out how you can use the data. If you are not sure, HM Land Registry recommends that you check with Royal Mail: example@royalmail.com.

How to get this data by visiting your local HM Land Registry office

Start by finding your nearest HM Land Registry Office.

When you visit your nearest office, bring:

  • ID 1
  • D 2

A member of staff will run security checks on you and share the data you need in a paper format.

How to get this data by post

Print the application form for data requests and fill it out.

When you post this form, include:

  • ID 1
  • ID 2

Post the completed form and copies of your identification to HM Land Registry:

HM Land Registry
Citizen Centre
PO Box 74
Gloucester
GL14 9BB

If HM Land Registry can verify you from your identification, we will post the printed data to you.
