HM Land Registry

Use land or property data

HM Land Registry publishes big data files about land and property on this page.

Most of this data is free except for the national polygon service and everyone can use it.

You can download some of this data without opening an account, but if you want to download licensed data (data that can only be used in a certain way), you will need to create an account first.

  1. See the data I can download without an account
  2. See the data I cannot download unless I create an account

All of these files show data:

  • organised by theme, for example ‘companies that own property’
  • as tabular (spreadsheet) files or shapefiles (shapes on a map)
