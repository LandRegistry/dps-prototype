Use land or property data

You can use this service to:

  • see a list of data (files) published by HM Land Registry
  • check how you can use the data
  • get your data licence (you need this to download licensed data)
  • download data files


National Polygon Service

Paid for

Zip file. inc. shape file & CSV, 7.3Gb

API available

Updated on the second day of every month

This data includes polygons, title number, estate interest, date of creation, date of update (where available), UPRN (Unique Property Reference Number – where available) and more.


It costs £20,000 a year, plus VAT. You can get this data by contacting HM Land Registry (find out how to do this on the next page).
