HM Land Registry data: sign in or create an account

It is free to create an account for HM Land Registry’s licensed data and it takes about 5 minutes to apply. Once you have created your account, you can download all of the licensed data files.

If you’re using the data for personal use or for a non-UK organisation, you can start downloading data immediately after creating your account.

If you are using the data for a UK organisation, HM Land Registry can take up to 2 days to approve your account while we run security checks. In most cases, your account will be approved more quickly.

Before you create an account, check the data licences to find out how you can use this data.

Start now

Before you start

Information you need to open an account

To create an account, you’ll need to tell HM Land Registry your name, email, address and phone number. You’ll also need to tell us if you’re a UK resident.

If you’re using the data as part of your work for a UK organisation, you’ll also need to include the name, address, type and registration number of the organisation you work for.

If you’re working for a non-UK organisation, you’ll need to include the name, address and country of incorporation of the organisation you work for.

Why do I need to open an account?

HM Land Registry asks for this information because these data files have a licence, conditions of access and/or a fee. You must agree to the licence for each data file before you can download it.

Free data and data with an access fee

You do not need to pay anything to open an account and most of the licensed data is also free.

There are a small number of data files with a fee. You can see the list of the data files with a fee (and their costs) on HM Land Registry’s fees page.

Consequences of data misuse

Data files with a licence have usage restrictions (you can use the data, but you can only use it in a certain way).

HM Land Registry asks you to create an account because of the legal consequences for data misuse. For example, if you sell the data HM Land Registry can share your account details with Royal Mail and you may be taken to court.

! Warning If you do not use the data according to the licence, you may be taken to court for data misuse.

Read each data licence to find out how you can use the data. If you are not sure, HM Land Registry recommends that you check with Royal Mail: example@royalmail.com

How to get this data by visiting your local HM Land Registry office

Start by finding your nearest HM Land Registry Office.

When you visit your nearest office, bring:

  • ID 1
  • ID 2

A member of staff will run security checks on you and share the data you need in a paper format.

How to get this data by post

Print [DO WE HAVE A PAPER APPLICATION FORM?] and fill it out.

When you post this form, include:
