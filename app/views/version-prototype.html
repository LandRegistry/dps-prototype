How to version the prototype

To version the prototype:

  1. Copy prototype folder
  2. Update the index file to link to new start / first page
  3. Go to login-fromstart.html and update hardcoded link the references the sprint
  4. If new sprint, add link the newest sprint tab on the design tracker and update URL
  5. Update links on all email temnplates
