About the API

Developers can access dataset with via the API as an alternative to the using the web interface for this service. The API provides both RESTful and functional interfaces, all in JSON format, making it suitable for a wide range of clients.

What you can access

Not all of our datasets are available through our API access. Datasets that have [xxxx] licence attached cannot be accessed using the API.

API access is however available for following datasets:

  • Overseas companies that own property in England and Wales
  • UK companies that own property in England and Wales
  • Registered leases

How to get access and generate API key

You need to have an account to access datasets using the API. If you havenâ€™t already created one, you can do so by clicking here.

If you have an account, you need to sign-in before you can generate an API key.

Once you have created an account and have signed-in, you can navigate to the API key generation screen.

After you have signed in, a link to generate an API key will appear in the admin menu.

By agreeing to both the licence agreement and the terms of use, you can generate an API key.

The base URL for the API is: https://something.gov.uk/api
