About the API

Software engineers can use this API to access datasets in the Use land and property data service. This is a RESTful API in JSON format, making it suitable for a range of programming languages.

What you can access with the API

You can use the API to get:

  • a list of HM Land Registry datasets that you can get from this service
  • a list of HM Land Registry datasets that are not part of the service
  • information and metadata about datasets
  • secure links for downloading datasets

You can access datasets that show:

  • Overseas companies that own property in England and Wales
  • UK companies that own property in England and Wales

How to access the datasets

You need to create an account and agree to a licence to access the datasets.

  1. Create an account if you don't have one
  2. Sign in to get your API key
  3. Agree to the data licence(s) on the Get land and property data website

Base URL

Access the API from the base URL: http://base-url/api/3/action/{action}

Full documentation
